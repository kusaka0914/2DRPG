【プログラミング言語】
- C++ (C++17)

【ビルドシステム】
- CMake

【グラフィックスライブラリ】
- SDL2 (ウィンドウ管理・入力処理)
- SDL2_image (画像処理)
- SDL2_ttf (フォント描画)

■ゲームの概要

【ゲームタイトル】堕天勇者と終焉の王都 -魔王に捧ぐ反逆の剣-

【ジャンル】RPG

【特徴】堕落した勇者が魔王の指示のもと街を滅ぼす物語です。戦闘システムとしては3すくみを採用しており、事前に3ターン分の行動を選択する形になっております。そして自分と相手の3ターン分のコマンドを比較して勝敗をジャッジし、3ターン後に勝数が多い方が一方的に敵を攻撃できるという非常に爽快感のあるゲームとなっております。また、夜の街に侵入して街の住民を次々と倒していきます。これを本作では住民戦と呼んでいます。住民戦では住民の行動予測をもとにして攻撃をするか、隠密をするかを判断して戦っていきます。さて、堕天勇者は街を滅ぼして魔王とともに最高の世界を作り上げることができるのでしょうか？

【プレイ時間】1時間〜程度

【ゲーム制作者】日下　拓海

■ゲームの起動

windows

本ページ右側にあるReleasesから最新版のFallenHeroAndTheDoomedCapital_win_v~.zipをインストールします。

そのzipファイルを展開し、「start_game.bat」ファイルをダブルクリックしてください。

mac

本ページ右側にあるReleasesから最新版のFallenHeroAndTheDoomedCapital_mac_v~.zipをインストールします。

そのzipファイルを展開し、「start_game.sh」ファイルをダブルクリックしてください。
